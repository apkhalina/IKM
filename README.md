## Основные сущности
- Книги (Book) - каталог библиотеки
- Читатели (Reader) - зарегистрированные пользователи
- Библиотекари (Librarian) - сотрудники библиотеки
- Выдачи (Loan) - выдача книг читателям (связующая таблица)

## Реализованный функционал
Для каждой таблицы:
- Просмотр данных - таблицы с пагинацией и поиском
-  Добавление данных - валидированные формы
- Редактирование данных - формы с предзаполненными данными
- Удаление данных - с подтверждением через диалог

## Валидация данных
- Проверка обязательных полей
- Уникальность идентификаторов (инв. номер, таб. номер и т.д.)
- Формат телефонных номеров (+7(XXX)XXX-XX-XX)
- Валидация дат и числовых диапазонов
- Проверка целостности данных при удалении
